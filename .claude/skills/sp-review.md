# Review Content Quality

## Instructions

When asked to review an article or content:

1. **Read the content** thoroughly
2. **Check structure:**
   - Has H1 title
   - Has abstract/intro paragraph
   - Uses H2/H3 hierarchy correctly
   - Has diagrams where architecture/flow would benefit
   - Tables for comparisons
3. **Check depth:** Is this Staff/Principal-engineer level? Not too basic?
4. **Check accuracy:** Technical claims, patterns described correctly
5. **Check completeness:** Missing sections, incomplete explanations
6. **Check frontmatter:** All required fields present and correct
7. **Report findings** with specific suggestions
