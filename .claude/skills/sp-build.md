# Build & Serve

## Instructions

1. **Run `bun run diagrams`** first (renders only changed diagrams)
2. **Run `bun run build`** to build the site
3. **If errors occur**, diagnose and fix them
4. **Report build results** (page count, tag count, build time)
5. **Optionally start dev server** with `bun run dev` if user wants to preview
